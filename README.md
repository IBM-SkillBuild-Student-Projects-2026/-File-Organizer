# 📂 File Organizer (Python)

A simple automation tool that organizes files into categories like:
- Images
- Videos
- Audio
- Documents

## How It Works
The script scans a folder and moves files into categorized folders based on extension.

## How to Run
1. Install Python
2. Run:
   python organizer.py
